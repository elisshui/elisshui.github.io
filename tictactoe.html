TIC-TAC-TOE

Figure 1. Tic-tac-toe game.

I coded the classic two-person Tic-Tac-Toe game using Python and pygame. The game features a simple graphical user interface to allow for ease of understanding as seen in the images.

As an extension to this project, I decided to create a "single-player" option. Selecting this option will guide the user to choose between an easy and hard level which they will then play. This extension is currently a work in progress.
