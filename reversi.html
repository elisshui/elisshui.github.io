Reversi

Figure 1. Game Starting Screen

I coded the classic Reversi game using C. The game allows the user to choose their playing mode (single player and two-players). If the single-player mode is selected, the user will be prompted to select their desired level of game difficulty, checker color, and the size of the board they would like to play on. Selecting the two-player mode will simply prompt the user to input the size of the chess board. The game's graphics are much simpler than thetic-tac-toe game I coded as my goal for this project was to improve my coding aptitude and logic.

When the hard level and a board size of 26 are selected (the largest size allowed), the average run-time of the AI algorithm is 0.621 ms while a board size of 4 (the smallest size) has an AI algorithm run-time of 0.271 ms. The videos below show the two playing modes. For the single-player mode, a board size of 8 was selected and the user is playing the easy level.

Error-checking functions were also implemented. During the game's set-up phase, if the user enters something outside of the game's constraints, they will be prompted to re-enter their desired input. For example, the board size must be an even number between 4 to 26 inclusive. Additionally, should the user input an invalid move, the game will immediately finish and the opponent will win.
