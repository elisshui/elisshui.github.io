Spotify Clone Website

This project was based on the Spotify clone project by JavaScript Mastery . I completed this project with the goal of improving my front-end development skills with React, Redux, and JavaScript. I also aimed at gaining more experience in working with and utilizing APIs. The website allows for exploring top artists and songs using data gathered live through Rapid API. It also lets users search for songs popular in their location through the IP Geolocation API. Features such as adjustable volume, playing, pausing, and viewing song lyrics were also implemented.

Tools used include: JavaScript, React, Redux Toolkit, Rapid API, and IP Geolocation API.
